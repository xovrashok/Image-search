Image Search Application

A modern, responsive web application for searching and viewing high-quality images using the Pixabay API. Built with vanilla JavaScript and bundled with Vite.

🚀 Features
Real-time Image Search: Fetch images dynamically based on keywords.

Asynchronous API Requests: Smooth HTTP requests handled via Axios with error catch boundaries.

Loading State: Visual loading indicator powered by pure-css-loader during API calls.

Interactive Gallery: Responsive image grid layout with smooth hover effects.

Lightbox Modal: Full-size image viewing with touch gestures, captions, and navigation using SimpleLightbox.

Notifications: Elegant, non-blocking toast alerts for empty inputs or empty search results via iziToast.

🛠️ Tech Stack & Libraries
Bundler: Vite

Language: JavaScript (ES6+), HTML5, CSS3 (Modern Flexbox/Grid, CSS Variables)

HTTP Client: Axios

UI/UX Components: SimpleLightbox, iziToast, pure-css-loader
